# Tebet

一款基于[Mirai Console](https://github.com/mamoe/mirai)的QQ机器人插件，TechBedrock服务器自用。

辣眼代码慎用！ 期待有路过大佬提供建议awa

计划会与Minecraft服务器有更多很**刺激**的联系，但是目前还没有想好怎么做。 =￣ω￣=

![机器酱](http://techbedrock.cn:81/storage/img/tebet2.jpg "Fischlandaz画的机器酱酱")

## 功能

1. 暴躁模式（私聊特供）：输入`启动暴躁模式`或`关闭暴躁模式`即可让Tebet暴躁/不暴燥,暴躁时候的Tebet会更**那啥**(=￣ω￣=)。

## 未来计划

1. 能获取Minecraft服务器的服务器信息（在线人数，服务器版本，服务器IP等），并通过ChatGPT生成。
2. 能获取Minecraft服务器的玩家信息（玩家名，玩家UUID，玩家等级，玩家经验等），并通过ChatGPT生成。
